SSM Metrics

Application metrics are used to better enhance this application (SSM) and give the user (You) a reliable experence. Information is sent to a secure Metrics and Analytics server.

The following information would be sent:

  • IP Address
  • IP GeoLocation
  • Application Starts
  • Application Stops
  • Error Messages

You may choose to accept or reject the capture of Satisfactory Server Manager Metrics using the buttons below.

If you accept and later decide to reject the metrics and request to remove all information about you, please contact metrics@hostxtra.co.uk
